## Local Preview

From the repo root:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://127.0.0.1:8000/
```

## Deployment

This repository is configured for GitHub Pages. Any committed changes to the published branch will be served as a static site.

Before pushing, run a quick static check:

```bash
git diff --check
```

Then commit and push normally.
